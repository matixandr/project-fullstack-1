# Project Guidelines

## Basic information

- Project name: CRYPTO.AI

### Techstack

- Next.JS 16.1.6
- Tailwind 4.0
- TypeScript 5
- Recharts (for market data visualization)
- Lucide React (for iconography)
- Cloudflare Turnstile (CAPTCHA)

### Typography

- Inter / Sans-serif

### Colorscheme (Dark Mode)

- Background: #09090b
- Primary: #3b82f6 (Blue)
- Success: #10b981 (Emerald/Green)
- Destructive: #ef4444 (Red)
- Card: #18181b

## Main project idea

The project is an AI-powered cryptocurrency trading dashboard. It features a "fake" automated trading agent that executes virtual trades based on real market data. Users can monitor the agent's performance, view market charts, and adjust trading strategies in a dark, professional interface.
